# Mario Mojica Homepage (mariomojica.com)

Este es el proyecto principal de la página web (landing page) de **[mariomojica.com](https://mariomojica.com)**.

Este proyecto es una aplicación web construida con [Next.js](https://nextjs.org/) a partir de `create-next-app`.

## 🚀 Cómo empezar

Primero, ejecuta el servidor de desarrollo:

```bash
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) (o el puerto que se te haya asignado, ej. 3004) en tu navegador para ver el resultado.

Puedes empezar a editar la página modificando `app/page.tsx` (o `src/app/page.tsx`). La página se actualizará automáticamente a medida que edites el archivo.

Este proyecto utiliza [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) para optimizar y cargar automáticamente [Geist](https://vercel.com/font), una nueva familia tipográfica de Vercel.
